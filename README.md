# Custom Electric Guitar Engineering Project

## Project Overview

This project documents the complete design and construction of a custom solid-body electric guitar. My goal is to learn woodworking, electronics, instrument design, and engineering by building a fully functional guitar from raw materials.

This project is being documented as part of the Hack Club Macondo hardware program.

---

## Objectives

- Design and build a solid-body electric guitar
- Learn woodworking techniques
- Install and set up a truss rod
- Build and shape the neck
- Install frets and hardware
- Wire the guitar electronics
- Set up the instrument for good playability
- Document every stage of the build

---

## Features

- Custom body design
- Bolt-on neck
- 24.75-inch (628 mm) scale length
- Single humbucker pickup
- One volume control
- One tone control 
- Adjustable bridge
- Wooden body with clear finish
